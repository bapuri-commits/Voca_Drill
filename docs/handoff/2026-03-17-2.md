## 핵심 결정 & 맥락

- **localStorage 직접 토큰**: authFetch 주입 방식에서 race condition 발생 → api.js가 매 요청마다 localStorage에서 직접 읽도록 변경
- **PDF Day 분할**: 300MB 전체 PDF → Day별 ~5MB + Test별 ~1MB, 기본 Day별 보기 + 전체 옵션
- **nginx 프록시**: `/syops-auth/` → SyOps API (CORS 우회), 프로덕션/로컬 환경 분기 (`import.meta.env.DEV`)

## 핵심 결정 & 맥락

- **quality 매핑**: 4단계 버튼(0-3) → SM-2 표준(0-5) 내부 변환. 모름=0, 헷갈림=2, 알겠음=4, 완벽=5
- **selectinload 채택**: joinedload는 LIMIT과 충돌 → selectinload로 통일
- **first_studied_at 추가**: updated_at만으로는 "오늘 처음 배운 단어"와 "오늘 복습한 단어" 구분 불가 → 별도 필드
- **resume_session 보류**: 현재 비기능적 (세션 원본 단어 목록 미저장). Phase 2에서 재설계
- **워크스페이스 모드 제약 무시**: 사용자 요청으로 워크스페이스에서 직접 개발 진행
